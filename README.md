## *install*

Install ruby and jekyll using the following instructions: https://jekyllrb.com/docs/installation/

Install the bundle:

```shell
bundle install
```

## *development*

Launch a local server to test the site with:

```shell
bundle exec jekyll serve
```

Then go to: http://127.0.0.1:4000/
